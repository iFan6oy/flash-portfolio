Cutt It is a premium barbershop booking app built as a real, co-founded venture with
a signed partnership behind it. It pairs a polished React Native client with a
Node/PostgreSQL backend and Square payments.

## Stack

- **Mobile:** React Native / Expo SDK 54, React Navigation, Reanimated 4, Lottie,
  expo-notifications.
- **Backend:** Express + PostgreSQL with JWT auth and bcrypt, Square SDK for payments
  and catalog sync, node-cron for scheduled jobs.

It's the strongest proof that I ship full-stack mobile products end to end — auth,
payments, scheduling, and a designed UI — for paying clients.
